WebPh.one beta is an open source progressive web application that allows you to make calls to Community Cellular Networks and to each other. It is developed by SayCel and Rhizomatica.

Webph.one provides you a virtual number and you can directly call PearlCel, the community mobile network in Pearl Lagoon, Nicaragua.

Use google chrome on any device, phone or laptop. On android download it to your home screen.

Webph.one is open source, can be configured to call any SIP network, and is free to use, modified and contributed by anyone.
